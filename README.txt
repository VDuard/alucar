Repositório para projeto de engenharia de software 2 (alucar)

Cada programador tem sua pasta para implementação de arquivos javascript para aprendizado
Suas pastas ficam em ...\alucar\js\aplicationTesters\SeuNome
Fiquem a vontade para modificar qualquer documento contanto que esteja em sua pasta

Os arquivos finais referentes a aplicação web serão colocados na pasta 
...\alucar\js\aplicationScripts\Script.js
Favor não adicionar apenas arquivos revisionados e aprovados pelo grupo nesta pasta
